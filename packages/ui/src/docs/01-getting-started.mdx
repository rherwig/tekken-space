# Getting Started

This is the style-guide for <a href="https://tekken.space">TekkenSpace</a>.
It contains colors, icons and components as well as information on how to use them.

## Installation
In order to use the elements from this library, install it via:

```bash
npm install @tekken-space/ui
```

Please note that this is currently only possible for projects inside the TekkenSpace mono-repo,
as this library is not currently published.
